# ABSTRACT

Dysarthria is a motor speech disorder characterized by weakness or incoordination of the speech musculature, resulting in reduced intelligibility and altered prosody. Early detection and objective assessment are critical for timely clinical intervention and rehabilitation monitoring. Traditional dysarthria assessment relies on subjective clinical evaluation, which is time-consuming, dependent on clinician expertise, and lacks standardized quantifiable metrics. This project presents an automated machine learning system for objective dysarthria classification using advanced speech feature extraction and ensemble learning methods.

A dataset comprising 2,104 audio samples (16 kHz sampling rate) was collected from dysarthric (n=1,569) and non-dysarthric (n=535) speakers. To address dataset imbalance and improve model generalization, data augmentation was applied using three complementary techniques: AddGaussianNoise (amplitude 0.001–0.015), TimeStretch (rate 0.8–1.25), and PitchShift (±3 semitones), resulting in a balanced augmented dataset of 4,208 samples. Two parallel feature extraction pipelines were implemented: (1) Wav2Vec2 pre-trained deep learning embeddings extracting 768-dimensional learned representations, and (2) acoustic feature engineering extracting 246 hand-crafted features including Mel-Frequency Cepstral Coefficients (MFCC, 40 coefficients), first and second-order derivatives (80 features), Root Mean Square energy, zero-crossing rate, and spectral centroid. Feature selection using SelectKBest with f_classif scoring reduced dimensionality to 150 optimal features, focusing on the most discriminative acoustic properties.

Three machine learning models were trained on 80% of the dataset and validated on a 20% held-out test set using stratified random splitting: Random Forest (200 estimators with balanced class weights), XGBoost (200 boosting rounds, learning rate 0.1, max depth 6), and MLP Neural Network (hidden layers 256–128, max iterations 300 with early stopping). Model performance was evaluated using accuracy, AUC-ROC, sensitivity, and specificity metrics, with confusion matrices generated for detailed error analysis.

Results demonstrated exceptional classification performance. Both Random Forest and XGBoost achieved 99.88% accuracy with AUC scores of 0.999, sensitivity of 99.7%, and specificity of 98.8%. The MLP achieved 99.64% accuracy. Confusion matrices revealed minimal misclassification: 0–1 false negatives (dysarthric cases missed) and 2–3 false positives across 842 test samples. Supplementary intelligibility analysis using Wav2Vec2-CTC phoneme transcription revealed mean confidence scores of 98.03% for dysarthric speakers versus 97.46% for non-dysarthric speakers, suggesting the models capture subtle prosodic and articulatory differences beyond surface intelligibility.

This study demonstrates that machine learning models trained on augmented acoustic features achieve near-perfect dysarthria classification suitable for clinical deployment. The high sensitivity (99.7%) ensures minimal false negatives, reducing risk of missed dysarthria cases. XGBoost is recommended for production due to its balance of interpretability, computational speed, and performance. The proposed system enables objective, rapid speech assessment applicable to clinical screening, remote patient monitoring, rehabilitation progress tracking, and neurological disorder assessment. Future work includes speaker-independent cross-validation, multi-severity classification, and validation on novel patient populations and additional neurological conditions.

**Keywords:** dysarthria, speech analysis, machine learning, Wav2Vec2, MFCC, acoustic features, XGBoost, automated classification, deep learning, speech pathology
